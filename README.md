# Rishabh Kumar Tiwari — Portfolio

A modern, dark-themed personal portfolio built with **React + Vite**, featuring glassmorphism design, scroll-triggered animations, and full responsiveness.

![React](https://img.shields.io/badge/React-19-61DAFB?logo=react&logoColor=white)
![Vite](https://img.shields.io/badge/Vite-8-646CFF?logo=vite&logoColor=white)
![License](https://img.shields.io/badge/License-MIT-green)

## ✨ Features

- 🌑 Dark premium theme with gradient accents
- 🪟 Glassmorphism cards with subtle hover effects
- ⌨️ Typewriter animation for dynamic role display
- 📜 Scroll-triggered reveal animations
- 📱 Fully responsive design (mobile → desktop)
- 🔗 Live social links (GitHub, LinkedIn, Email)
- ⚡ Lightning-fast with Vite

## 🛠️ Tech Stack

- **Frontend**: React 19, JavaScript (JSX)
- **Build Tool**: Vite 8
- **Styling**: Vanilla CSS with CSS variables
- **Icons**: Lucide React
- **Animations**: CSS transitions + Intersection Observer
- **Fonts**: Inter, JetBrains Mono (Google Fonts)

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/rishabhtcodes/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## 📁 Project Structure

```
src/

│   ├── Achievements.jsx  # Hackathon & recognition cards
│   ├── Contact.jsx       # Get in touch CTA
│   └── Footer.jsx        # Copyright & socials
├── App.jsx
├── main.jsx
└── index.css             # Design system & global styles
```
